\doxysubsection{Detailed Description}
\begin{DoxyVerb}This class represents a complete extracted route sequence as output by the ICAO Field 15 Parser.
This class stores 'n' ExtractedRouteRecord instances that together describe a complete route.
\end{DoxyVerb}
