\doxysubsection{Detailed Description}
\begin{DoxyVerb}Enumeration to identify the message type, ICAO ATS, OLDI, ADEXP or UNKNOWN; for OLDI
messages the field content for a given title can vary depending on the adjacent unit that
a given OLDI message is exchange on. Using a combination of the message type and
adjacent unit a precise field list can be obtained for a given message title.\end{DoxyVerb}
