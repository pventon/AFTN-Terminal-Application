\doxysubsection{Detailed Description}
\begin{DoxyVerb}This class contains a dictionary of error messages used by the ICAO Message Parser
when erroneous fields and / or subfields are detected.

The error message dictionary keys are defined in EnumerationConstants.ErrorId class.\end{DoxyVerb}
